Welcome to Plugin Playground

Plugin Playground is an open-source runtime tweak subsystem for macOS Apple Silicon.

It provides a kernel-level interception engine — built on Mach exception ports and ARM64 hardware breakpoints — for building plugins, introspection tools, and behavior-modification tweaks that work on any process without SIP workarounds.

Includes

bin/fangs — reference tweak / tracer for launchd
bin/grant — amfid privilege helper
tweaks/ — user-owned directory for tweak bundles
